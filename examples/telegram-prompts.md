# Telegram Prompt Examples

## Clarification-first

- 이거 애매한 부분부터 다 물어보고 시작해
- 구현 전에 요구사항부터 정리해줘
- 내가 뭘 원하는지 먼저 캐봐

## Planning-first

- 바로 코딩하지 말고 계획 먼저 잡아
- 합의 가능한 구현안부터 만들어줘
- 리스크 큰 작업이라 설계 먼저 보자

## Execution-first

- 이 계획대로 끝까지 밀어
- 묻지 말고 진행 가능한 데까지 진행해
- 구현하고 검증까지 해줘

## Parallel/team style

- 이건 나눠서 병렬로 처리해줘
- 워커 여러 명 돌리는 느낌으로 해봐
- 큰 작업이니까 분업해서 밀어

## QA / cleanup

- 테스트 깨진 거 다 잡아줘
- 빌드, 린트, 테스트까지 마무리해
- 검증 루프 돌려서 끝내

## Resume / interruption

- 아까 그거 이어서
- 멈췄던 작업 재개
- 지금 상태 기준으로 계속
- 여기서 멈추고 요약만

## Maximum-autonomy wording

- 완전 알아서 해봐
- madmax 느낌으로 끝까지 밀어
- autonomy 모드처럼 진행해
